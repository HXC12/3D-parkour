# 3D-parkour
3D是一款跑酷类游戏，主要考验用户的反应速度，具有快速活跃的游戏节奏，用户操作小球躲避障碍物，吃取星星，赢取得分，挑战最高记录。
用到的库有openGL,assimp读取obj库，std_image读取读取图形库等。

操作：利用上左右键来控制小球的移动。

暂停：k键
